Match Madness!

How to Play:

Flip the cards to reveal hidden images. Find two cards with the same image to score points!

Challenge:

  • Match all the pairs before time runs out!
  • You've got 30 seconds on the clock!

Scoring:

  • Each correct match earns you 100 points!
  • Watch out for mistakes! Wrong guesses cost 50 points.
